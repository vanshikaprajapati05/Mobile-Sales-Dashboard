# 📱 Mobile Sales Dashboard – Power BI Project

## 📌 Project Overview
This project is an interactive Mobile Sales Dashboard created using Power BI.  
The dashboard helps analyze mobile phone sales across different cities, brands, payment methods, customer ratings, and monthly trends in India.

It provides a clean visual representation of sales performance and customer behavior using charts, maps, KPIs, and filters.

---

## 🚀 Features
- 📊 Total Sales, Quantity & Transactions Overview
- 🗺️ Sales by City using Map Visualization
- 📈 Monthly Sales Trend Analysis
- 📱 Sales by Mobile Model
- 💳 Transactions by Payment Method
- ⭐ Customer Ratings Analysis
- 📅 Day-wise Sales Performance
- 🔍 Interactive Filters/Slicers:
  - Brand
  - Mobile Model
  - Payment Method
  - Day Name
  - Month

---

## 🛠️ Tools & Technologies Used
- Microsoft Power BI
- Microsoft Excel / CSV Dataset
- Data Cleaning & Visualization
- DAX Measures
- Interactive Dashboard Design
---

## 📈 Key Insights
- Total Sales reached 769M
- Highest sales observed during March and July
- UPI was the most preferred payment method
- Major sales contributions came from metropolitan cities
- Apple and Samsung showed strong sales performance
